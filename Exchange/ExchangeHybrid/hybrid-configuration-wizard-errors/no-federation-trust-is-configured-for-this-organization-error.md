# No Federation trust is configured for this organization error when you run the Hybrid Configuration Wizard

_Original KB number:_ &nbsp; 4015612

## Symptoms

When you run the Hybrid Configuration Wizard (HCW), you receive the following error message:

> Set-FederatedOrganizationIdentifier is failing with error "No Federation trust is configured for this organization or no domain is federated as the account namespace."

## Cause

This issue occurs if the tenant forest moves within the Microsoft Data Center.

## Resolution

To resolve this issue, open [a support case](https://support.microsoft.com/contactus/) and have the issue fixed by Microsoft. There are no actions that an organization administrator can take to resolve this issue.
