# Public folder meetings cannot be modified in OWA when you use Outlook on the web to change a meeting

_Original KB number:_ &nbsp; 3148910

## Problem

When you try to change a meeting in a public folder by using Outlook on the web (formerly called Outlook Web App), you receive the following error message:

> Error: Public folder meetings cannot be modified in OWA.

## Cause

It's unsupported to use Outlook on the web to edit meetings that contain one or more attendees in public folders. However, using Outlook on the web to edit appointments in public folders is supported.

## Solution

To make changes to meetings in public folders, use Microsoft Outlook.
