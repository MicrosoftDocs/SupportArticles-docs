# Admins cannot add guest users to their organization in Microsoft 365 admin center

## Symptoms

You cannot add guest users to your organization by using the Microsoft 365 admin center. 

## Workaround

To work around this issue and add the guest user, go to the [Azure Active Director (AAD) admin center](https://aad.portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/AllUsers?WT.mc_id=365AdminCSH), and then select New guest user. 

## More information

For more information about how to add new users, see [Quickstart: Add guest users to your directory in the Azure portal](https://docs.microsoft.com/eazure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).
