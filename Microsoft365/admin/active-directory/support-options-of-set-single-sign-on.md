# Support options for setting up single sign-on for Office 365

_Original KB number:_ &nbsp; 2966531

This article lists support options that are available for setting up single sign-on (SSO) for Office 365.

Office 365 includes technical support. However, the support option to implement SSO or to change SSO configuration for Office 365, is not included with the Office 365 subscription. The other options available for these tasks are as follows.

## No-cost technical guidance

Here is a free step-by-step guide to complete Active Directory Federation Services (AD FS) implementation tasks in a self-service for Office 365 customers:

- Step-by-step implementation guidance: [Checklist: Use AD FS to implement and manage single sign-on](https://docs.microsoft.com/previous-versions/azure/azure-services/jj205462(v=azure.100)?redirectedfrom=MSDN)

If you have questions about this step-by-step guide, additional resources are available from the Office 365 community:

- Directory Integration Services discussion forum in the [Office 365 community](https://techcommunity.microsoft.com/t5/office-365/ct-p/Office365)
- Directory Integration Services wiki article in the [Office 365 community](https://docs.microsoft.com/office365/enterprise/office-365-integration)

## Paid technical advisory services

To request professional support over the web, follow these steps:

1. Go to the [support request page](https://support.microsoft.com/assistedsupportproducts), and then select Office Products.
1. In the list of products, select the Office 365.

    :::image type="content" source="./media/support-options-of-set-single-sign-on/support-list.png" alt-text="list of products":::

1. Select the link [Microsoft 365 Admin Center](https://admin.microsoft.com/).
1. Sign in with your Microsoft account.
1. Select **Support** options.
1. In **Problem type**, select **Active Directory**.
1. In the next Category, select **Active Directory Federation Services (AD FS)**.
1. Start **Request**.
