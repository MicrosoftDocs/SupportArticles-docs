# "Cannot be opened" error when you start an Office Starter 2010 program

## Resolution

If Word Starter 2010 or Excel Starter 2010 can't be opened, uninstall and reinstall Microsoft Office Starter 2010: 

1. Select Start (or Start > Run in Windows XP).    
2. Type appwiz.cpl, and then press Enter.

   **Warning** Don't remove Microsoft Office 2010 from the list of installed programs. You'll need this to reinstall Microsoft Office Starter 2010.

3. Select Microsoft Office Starter 2010, and then select Uninstall (or Remove in Windows XP).

    :::image type="content" source="media/cannot-be-opened-error-when-start/uninstall-program.png" alt-text="Screenshot to select Uninstall after selecting the Microsoft Office Starter 2010 program.":::
4. Select Yes and follow the instructions to uninstall Office Starter 2010.

    :::image type="content" source="media/cannot-be-opened-error-when-start/click-to-run-application-manager.png" alt-text="Screenshot to select the Yes option in the Click-to-Run Application Manager window." border="false":::
5. To reinstall Office Starter 2010, select Start > All Programs > Microsoft Office 2010.

    :::image type="content" source="media/cannot-be-opened-error-when-start/start-menu.png" alt-text="Screenshot to select Microsoft Office 2010 in the Start menu." border="false":::
6. Select Use.

    :::image type="content" source="media/cannot-be-opened-error-when-start/use-office-2010.png" alt-text="Screenshot to select the Use option in Microsoft Office 2010." border="false":::
7. And then select Open. This will reinstall Office Starter 2010.

    :::image type="content" source="media/cannot-be-opened-error-when-start/open-office-2010.png" alt-text="Screenshot to select the Open option in Microsoft Office 2010." border="false":::
 If the error continues to occur, you may have to do a full restore of the computer to reinstall Microsoft Office Starter 2010. We recommend you contact the computer manufacturer to talk about your options before you do this.
