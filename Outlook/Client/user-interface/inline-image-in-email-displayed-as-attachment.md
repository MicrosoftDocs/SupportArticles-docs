# Inline image in an email is shown as an attachment when you open the message in Outlook 2013 or 2010

_Original KB number:_ &nbsp; 978814

## Symptoms

You receive an e-mail message that has an inline image. When you open the e-mail message in Microsoft Outlook 2010 or Outlook 2013, the image is displayed as an attachment.
