# You can't share files with external users from an Office 365 group site

## Problem

Consider the following scenario:

- You browse to the OneDrive page for an Office 365 group.

- You click a document or folder and then click **Share**.

- When you type the account for an external user, you receive one of the following error messages:

    - **We couldn't find an exact match.**
    
    - **Sorry, you are not allowed to share this with external users.**

## More information

This issue occurs because sharing files with external users from an Office 365 group site isn't supported.
