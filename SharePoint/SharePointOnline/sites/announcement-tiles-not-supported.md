# "This feature is not supported" when you activate the Announcement Tiles feature in a site

## Problem

Consider the following scenario:

- You browse to the Site Settings page in Microsoft SharePoint Online.

- You click Manage site features.

- In the Announcement Tiles feature, you click Activate.

In this scenario, you receive the following error message:

**This feature is not supported in the current site collection's version.**

## More information

This error occurs because the Announcement Tiles feature is available only on the My Site host on which the My Sites websites reside.
