# "Unable to find specified web in the given URL" when you use a calendar overlay

## Problem

Consider the following scenario:

- You browse to a Microsoft SharePoint Online calendar.

- You click the **CALENDAR** tab and then click **Calendars Overlay**.

- On the **Calendar Overlay Settings** page, you click **New Calendar**.

- You enter the URL of a SharePoint Online calendar in the **Web URL** field.

- You click **Resolve**.

In this scenario, you receive the following error message: 

**Unable to find specified web in the given URL.**

## Solution

This issue occurs because calendar overlays from one SharePoint Online site collection to another aren't supported.
