# Unable to drag and drop web parts onto classic pages in SharePoint Online

## Summary

When you try to drag and drop a web parts within or across Web Part Zones using modern browsers on Classic Pages in SharePoint Online, the Web Part doesn't move.

## More information

As a workaround, use Internet Explorer 10 in document mode. For more information, see [Display a classic SharePoint Online site in Internet Explorer 10 document mode](https://support.office.com/article/Display-a-classic-SharePoint-Online-site-in-Internet-Explorer-10-document-mode-4b4572b7-9223-45ec-8497-557a643da12a).

Microsoft is researching this problem and will post more information as it becomes available.
