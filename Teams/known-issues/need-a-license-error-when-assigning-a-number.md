# Error "You need a phone system license" when assigning a toll service number from the admin center

## Symptoms

When you try to assign a toll service number in the Teams admin center, you see the error "You need a phone system license."

## Resolution

Use PowerShell cmdlets to assign a toll service number instead. 

## More information

For more information about assigning phone numbers to resource account, see [Manage resource accounts in Microsoft Teams](https://docs.microsoft.com/microsoftteams/manage-resource-accounts).
