# Notifications and preview issues using Microsoft Teams on iOS 13 Beta

## Summary

You might encounter one of the following issues using Microsoft Teams on iOS 13 Beta:

- Teams notifications are not activated. This includes chats, mentions, and calls. 
- The file preview function does not work with the beta build. 

## More information

At this time there is no workaround. We are working with Apple developers to find fixes for these issues.
