# Notifications and preview issues using Microsoft Teams on iOS 13 Beta

## Summary

You might encounter one of the following issues using Microsoft Teams on iOS 13 Beta:

- Teams notifications aren't activated, including chats, mentions, and calls.
- The file preview function doesn't work with the beta build.

## More information

At this time, there's no workaround. We're working with Apple developers to find fixes for these issues.

Still need help? Go to [Microsoft Community](https://answers.microsoft.com/).
