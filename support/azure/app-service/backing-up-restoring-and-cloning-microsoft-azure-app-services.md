# Back up, restore, and clone an Azure App Service application

## Summary

This video demonstrates how to back up, restore, and clone your Azure App Service application. First, you learn how to make a manual or scheduled backup of your app. Then, you learn the procedures, limitations, and requirements for restoring the app from a traditional backup or from a *snapshot*. Finally, you learn how to clone an existing web app to a newly created web app in an App Service Plan in a different region.
