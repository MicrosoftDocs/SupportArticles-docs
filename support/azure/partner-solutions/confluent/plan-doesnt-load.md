# Troubleshoot offer plan doesn't load in Azure Marketplace for Confluent Cloud

This article helps you troubleshoot issues where the offer plan for Confluent Cloud doesn't load in Azure Marketplace. This error is often intermittent and may resolve by retrying.

## Prerequisites

- Access to the Azure portal and permission to deploy resources.
- Browser access to Azure Marketplace.

## Potential quick workarounds

1. Refresh the Azure portal and try to deploy the offer again.
2. Try using a different browser or an incognito/private window.
3. Clear browser cache and cookies.

## Troubleshooting checklist

### 1) Retry deployment

- If the offer plan doesn't load, refresh the page and try again.
- Wait a few minutes before retrying, as the issue may be intermittent.

### 2) Try a different browser or session

- Use a different browser or open an incognito/private window to rule out caching issues.

### 3) Clear browser cache

- Clear your browser cache and cookies, then retry the deployment.

### 4) Escalate to support

- If the issue persists after multiple attempts, contact [Confluent support](https://support.confluent.io) with details about the error and your environment.

## Causes and/or solutions

### Solution: Intermittent portal or browser issue

1. Retry the deployment after refreshing the page or using a different browser.
2. Clear browser cache and cookies.

### Solution: Persistent issue

1. Collect error details and contact Confluent support for assistance.

## Advanced troubleshooting and data collection

- Collect the following information if you need to escalate to Confluent or Microsoft support:
  - Azure subscription ID
  - Browser and OS version
  - Screenshots of the error
  - Timestamp of the deployment attempt
