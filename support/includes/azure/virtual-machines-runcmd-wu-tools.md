> [!IMPORTANT]  
> **Virtual machine (VM)** running in Azure and experiencing Windows Update issues?
>
> Trying to diagnose and resolve Windows Update or Windows operating system upgrade issues for your Azure VM? Try using one of the following tools:
>
> - [Azure VM Windows Update Error Detection Tool](../../azure/virtual-machines/windows/windows-vm-ipu-tool.md) to diagnose specific Windows Update errors.
> - [Azure VM Windows Windows Update Reset Tool](../../azure/virtual-machines/windows/windows-vm-wureset-tool.md) to reset the Windows servicing stack.
> - [Azure VM Windows OS Upgrade Assessment Tool](../../azure/virtual-machines/windows/windows-vm-osupgradeassessment-tool.md) to validate the operating system upgrade path and any known issues.
>
> If you're having issues updating or upgrading your VM, run these tools first before you contact Microsoft Support.
